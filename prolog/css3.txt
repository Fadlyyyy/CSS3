penjelasan css 3

css 3 adalah generasi ke 3 dari spesifikasi css yang direkomendasikan oleh w3c.

fitur yang ada pada css 3 dinamakan dengan modul.

kenapa harus menggunakan css 3

1. membuat website kita menjadi fleksibel dan interaktif
2. membuat mock-up dengan cepat tanpa bantuan bahasa scripting
3.mengurangi penggunaan gambar supaya meningkatkan kecepatan load pada halaman
4. penggunaan selector untuk menghindari penggunaan markup yang boros,bahkan mungkin tidak lagi dibutuhkan id dan juga class 
alasannya adalah penggunaan id dan class membuat sebuah elemen menjadi terbebani.

ada beberapa properti yang hanya berjalan di beberapa browser tertentu atau browser support
